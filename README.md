# Fire Emblem: Three Houses — Guide (static site)

A browsable, self-contained **static-HTML guide** for *Fire Emblem: Three Houses*
(Nintendo Switch). Open **[`index.html`](index.html)** locally, or visit the
**GitHub Pages** site for this repository.

火焰纹章：风花雪月 攻略静态网站。打开 `index.html` 或本仓库的 GitHub Pages 即可浏览。

## What's inside

- **`facts/`** — **objective** game data (bilingual English + 简体中文):
  - `character/` — the three houses' students (Black Eagles, Blue Lions, Golden Deer): stats, growth rates, Crests, personal skills, proficiencies, recruitment.
  - `others/` — church/faculty, Ashen Wolves (DLC), Byleth, special recruits, and major NPCs.
  - `classes/` — every class: certification requirements, stat/growth bonuses, abilities, mastery.
  - `conversation.html` — optimal dialogue/support choices for the whole game.
- **`suggestions/`** — **opinionated** recommendations (English):
  - `character_recommendation/` — stack-ranked per-character builds (class paths, mastery detours, training, supports, loadouts).
  - `team_recommendation_overview.html` + `routes/` — a **four-playthrough plan** (Azure Moon → Crimson Flower → Silver Snow → Verdant Wind) with rosters, pairings, and per-route detail.
  - `walkthrough/` — the Azure Moon day-by-day walkthrough & training schedule, a monastery-activities reference, and a playthrough log.
  - `community-recommendations.html` — unique tips gathered per forum.

## Notes

- **Self-contained:** every internal link stays inside this site; no external/relative links escape it. A `.nojekyll` file is included so GitHub Pages serves all files as-is.
- **Objective vs. opinion:** `facts/` is verified game data; `suggestions/` is judgment/recommendation. They are kept separate on purpose.
- This repository is the **published output**. The authoring sources, the crawl cache, and the recommendation rules live in the parent project, not in this site repo.
