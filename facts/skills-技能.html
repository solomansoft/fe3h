Skills & Abilities / 技能与能力

Reference for the combat skills referenced throughout the builds: what each does and how you get it. / 培养建议中反复出现的战斗技能速查：效果与获取方式。

Acquisition is cross-checked against this repo's own class files (the per-class Mastery Ability / 精通技能 and Class Abilities / 职业技能 sections) and sources/serenes-forest/classes-abilities.html. Effects/ranks marked ⚠️ are not fully pinned by a cached source — verify before relying on exact numbers. / 获取方式以本仓库 兵种资料（各职业“精通技能/职业技能”）与缓存来源核对；带 ⚠️ 者精确数值待核实。

How skills are acquired / 技能获取途径

A unit's 5 equipped ability slots can be filled from four sources: / 单位的 5 个技能装备格可来自四类来源：

  1. Class Mastery / 职业精通 — fill a class's mastery bar to permanently learn its Mastery Ability (and/or Mastery Combat Art). These are the ⛏ "detour" skills in the builds — you must spend time in that class to harvest them. / 填满某职业的精通槽，永久习得其精通技能/精通战技（建议中的 ⛏ 绕路技能）。
  2. Class Ability / 职业技能 — innate while the unit is in that class (e.g. the Faires, Alert Stance, Movement +1). Lost if you leave the class, unless it's also masterable. / 处于该职业时自带（如各类 Faire、警戒架势、移动+1），离开即失效。
  3. Skill Rank / 技能等级 — learned at a weapon/Authority proficiency rank (e.g. Battalion Wrath at Authority C, the +Avo rank skills). Permanent once learned. / 达到武器/权威熟练度后习得（如权威 C 的营队之怒），永久。
  4. Personal / 个人技能 & Budding Talent / 天赋觉醒 — unique to a character (e.g. Mercedes' Live to Serve, Dimitri's Seal Movement via Riding). / 角色专属（如梅尔赛德丝“舍己为人”、蒂米特利经骑术觉醒的“封锁移动”）。
