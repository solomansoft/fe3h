Route 2 — Crimson Flower (Black Eagles → Empire) — Hard

Run 2 of 4. Side with Edelgard against the Church in Ch.11. Short, fast route; final maps punish ground units, so lean on fliers. Rules: ../team-rules.md · Overview: ../team_recommendation_overview.md Tags: 🔒 route-exclusive · ⚠️ tutor/special-care · 💃 dancer · ⭐ top carry · 🧩 DLC-optional (non-DLC alt given) · (core)/(flex) Per-unit deep detail: ../../character_recommendation/

House: Black Eagles (Empire) · Difficulty: Hard Byleth class this run (R3 — distinct): Wyvern Lord (飞龙领主) — first optimized run. 8 Mov + Flying is ideal for CF's ground-hostile final maps. Distinct from R1 (Mortal Savant), R3 (War Master), R4 (Enlightened One). All four Byleth classes differ. Byleth gender/birthday: either gender (Wyvern Lord is not gender-locked; Byleth × Edelgard S-support works for M and F on CF). Birthday is a free calendar pick — choose an empty week. Roster size: 13 with DLC (Anna) / 12 non-DLC.

🔒 Route-exclusive highlights (verified)

  • Edelgard — playable only on Crimson Flower.
  • Hubert — playable only here; leaves if you side with the Church.
  • Jeritza — free, NOT DLC (ships with the free 1.1.0 update). Auto-joins start of Part 2 (~Ch.13) as a Lv27 Death Knight, no recruitment.
  • Church units are NOT available here (Seteth, Flayn, Catherine, Cyril, Shamir) → featured in R3 instead.

Roster composition (R9 — built as a team)

  • Damage split: Well balanced. Magic: Hubert (Dark Knight), Dorothea (Gremory), Hanneman (Warlock, flex), Manuela (Holy Knight, flex) hit low-Res/armor. Physical: Edelgard, Byleth, Jeritza, Ferdinand, Bernadetta (bow), Caspar (brawl).
  • Healing/staff: Linhardt (Bishop, Warp/Physic primary) + Dorothea (Faith A → Physic) + Manuela (faculty healer, flex). Solid.
  • Fliers / mobility: Byleth, Jeritza, Edelgard (Wyvern/armored-flyer offense) give air; Ferdinand (Paladin) + Hubert/Manuela (mounted mages) give Canto. Linhardt's Warp repositions the slow brawler.
  • Ranged / effective coverage: Bernadetta (Sniper/Bow Knight, anti-flier) + Hubert/Dorothea magic (anti-armor) + 1–2 range from mages. CF's late maps reward fliers + bows — both present.
  • Gambits/Authority: Edelgard + Byleth carry the strongest battalions; Caspar wants a Stride/Warp escort for his low Move.
  • Composition picks flagged below: Ferdinand → Paladin (#1 anyway, but chosen over Wyvern to keep his R3 Wyvern distinct); Bernadetta runs Sniper→Bow Knight for mobility; Manuela/Hanneman flex composition slots deepen CF's thin caster bench.

Deployment roster (13 DLC / 12 non-DLC)

Byleth — sword/flying carry (core)

  • Gender & birthday (recommended): Male · Harpstring Moon (M5), wk 1. Edelgard romances both genders, so this is the one route the plan allows male Byleth — take it to experience male Byleth (and, if you prefer raw power, the male-only War Master instead of Wyvern Lord). Female works for Byleth × Edelgard too. Early M5 birthday → Black Eagle Pendant (Charm +2) in both parts.
  • This run's class (R3 — distinct): Wyvern Lord (#1). Myrmidon/Soldier → Brigand (Axe + Death Blow) → Wyvern Rider → Wyvern Lord. Needs Axe A + Flying A tutoring — worth it.
  • Mastery detour: Brigand → Death Blow; Archer → Hit+20.
  • Loadout: Sword of the Creator + Brave Axe; Swordfaire, Death Blow, Hit+20, Alert Stance+; Authority battalion + Retribution.

Edelgard 🔒 ⭐ — axe lord / armored carry (core)

  • Class: Noble → Fighter → Brigand → Armored Lord → Emperor (unique, #1) — Raging Storm + Aymr.
  • ⚠️ Flags: none — Imperial Lineage gives 1.2× EXP.

Hubert 🔒 ⚠️ — mounted dark mage (core)

  • Class: Noble → Monk → Mage → Dark Bishop → Dark Knight (#1 — only class with Dark Tomefaire; needs Riding A + Lance C).
  • ⚠️ Flags: needs Riding A tutoring (off-strength). Do NOT build him as a Reason Warlock.

Jeritza 🔒 ⚠️ — high-power killer / flier (core)

  • Class: unlock Flying budding → Wyvern Rider → Wyvern Lord (#1). Or → War Master (#2) / ride Death Knight (#3, zero-effort).
  • ⚠️ Flags: late-joiner (~Ch.13, enemy Death Knight class) — reclass immediately, build Axe/Flying.

Petra 💃 ⭐ — dancer (core)

  • This run's class (R3): Dancer (win the White Heron Cup) — top Spd + Cha + Sword dodge. With Edelgard/Jeritza/Ferdinand as flying carries, her best team role is the refresher.

Ferdinand — lance cavalry (core) (composition pick)

  • This run's class (R3): Noble → Soldier → Cavalier → Paladin (#1 — Swift Strikes + Move 8 + Canto). (Reused in R3 as Wyvern Lord, so Paladin is his distinct build here per R3.)

Dorothea — mage (core) (reuse-variant: Gremory)

  • This run's class (R3): Noble → Monk → Mage → Warlock → Gremory (her #1 build — doubles all spell & Physic uses). She was the Dancer in R1; this satisfies R3 (different class on reuse).

Bernadetta ⚠️ — mobile bow nuke (core)

  • ⚠️ Flags: frail / worst bases — feed Energy Drop/Seraph Robe and Str/Spd boosters; needs a tank to draw aggro. Devastating once fed.

Caspar ⚠️ — brawler crit DPS (core)

Linhardt — healer / utility (core)

💃 Dancer this route: Petra

Highest Speed in the cast plus strong Charm and Sword — textbook dance/dodge. With Edelgard, Jeritza and Ferdinand providing flying offense, Petra's refresher value beats her marginal combat value. Distinct from R1 (Dorothea), R3 (Manuela), R4 (Yuri / Marianne).

🪙 Stealer this route: Petra

Run one Thief-master to Steal rare items off enemies — Dark Seal, stat boosters, consumables — the only repeatable way to farm them. Petra is ideal: Sword/Bow boon + top Speed (Steal is Speed-gated — only foes with lower Spd). She already detours Assassin, so slot a Thief master first; mastery makes Steal equippable in her final class (no long-term cost). See team overview · facts: Thief.
