Sylvain Jose Gautier / 希尔凡 — Build Menu V2

Blue Lions. Auto-joins on Azure Moon; F!Byleth recruits him free from any house (Ch.2). Minor Crest of Gautier (40% Mt+5 on combat art). Personal: Philanderer (+2 dmg / −2 taken near a female ally). Well-rounded growths (55% HP / 45% Str / 50% Spd / 40% Def) → a durable mounted/flying bruiser. Owns relic Lance of Ruin (破裂の槍).
Proficiencies: Lance (boon), Axe (boon), Riding (boon); Bow (bane); budding Reason → Black Magic Avo+20; Sword/Brawl/Flying/Faith/Authority/HeavyArmor neutral.
Harvest note: Death Blow (Axe boon) is a free harvest. Bow bane → NO Hit+20 harvest via Archer; fix accuracy with an Accuracy Ring instead.
Sources: fe3h.com / Game8 / Reddit (EN), note.com / GameWith / watasonhakase JP, GamerSky CN.

Build 1 — Paladin (圣骑士)

Path: Cavalier → Paladin (terminal). Cert: Lance A (boon) + Riding C+ (boon) — all on-strength, light. Push Lance S for Lancefaire + Swift Strikes (Lance A).

Mastery detours (boon-harvest): Death Blow [⛏ Brigand, Axe C — boon harvest] "non-negotiable" — the Swift Strikes + Lance of Ruin kill combo. SKIP Hit+20 (Archer needs Bow C — Bow is his bane); use an Accuracy Ring for Lance of Ruin's poor accuracy. Desperation [Cavalier] optional.

5 skills: Lancefaire · Death Blow · Swift Strikes enabler · Desperation · Crit+10.

  • ✔ 8 move + Canto + Lancefaire straight off his boons — the lowest-effort strong build, no banes touched.
  • ✔ Swift Strikes + Death Blow + Lance of Ruin deletes enemies on player phase.
  • ✘ Lance of Ruin has poor Hit — needs the Accuracy Ring since Hit+20 is bane-gated.
  • ✘ Less raw mobility/ceiling than the wyvern.

Build 2 — Wyvern Lord (飞龙领主)

Path: Brigand (master) → Wyvern Rider → Wyvern Lord. Cert: Lance C / Axe A (boon) / Flying A (neutral). His Lance/Axe/Riding kit certifies it easily.

Mastery detours (boon-harvest): Death Blow [⛏ Brigand, Axe C — boon, on path] free harvest. SKIP Hit+20 (Bow bane) — Accuracy Ring instead.

5 skills: Lancefaire (or Axefaire) · Death Blow · Swift Strikes enabler · Defiant Crit · Alert Stance+.

  • ✔ 8 move + flight + Canto — highest ceiling; Swift Strikes + Lance of Ruin from the air.
  • ✔ His durable growths make a fearless flier; Death Blow still free-harvested.
  • ✘ Needs Flying A (neutral) — more setup than Paladin.
  • ✘ Same Bow-bane accuracy issue; needs the Accuracy Ring.

Build 3 — Great Knight (重装骑士) — niche bulk

Path: Cavalier → Paladin (master) → Great Knight. Cert: Lance A (boon) + Riding (boon) + HeavyArmor A (neutral — steep). Pairs Lance of Ruin's Lancefaire with armored bulk [Game8].

Mastery detours (boon-harvest): Death Blow [⛏ Brigand, Axe C — boon]. SKIP Hit+20 (Bow bane).

5 skills: Lancefaire · Death Blow · Swift Strikes enabler · Weight-3 · Crit+10.

  • ✔ Tanky lance cavalry — armored frame + Lancefaire + 7 move.
  • ✘ HeavyArmor A (neutral) is steep; high skill demands for marginal gain [pegasusknight JP].
  • ✘ Paladin/Wyvern out-value it for less investment.
