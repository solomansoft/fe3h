Ferdinand von Aegir / 菲尔迪南特 — Build Menu V2

Black Eagles student. Stays on Crimson Flower; auto-joins Silver Snow; recruitable to other houses (Heavy Armor or Dexterity stat). Minor Crest of Cichol, balanced growths (HP 50% / Str 45% / Spd 50% / Cha 40%), personal Confidence (Hit/Avo +15 at full HP).
Proficiencies: Sword (boon), Lance (boon), Axe (boon), Riding (boon); no bane; budding talent Heavy Armor (Seal Speed). Authority/Bow/Brawl/Reason/Faith/Flying neutral.
Gender note: Darting Blow comes only from Pegasus Knight (female-only) — Ferdinand is male, so that detour is unavailable; raise Speed via boosters/Cavalier instead.
Sources: Game8 / GameFAQs / fe3h.com (EN), note.com / Pegasusknight (JP), GamerSky (CN).

Build 1 — Paladin (圣骑士)

Path: Noble → Soldier → Cavalier → Paladin. Cert: Lance B + Riding B — both boons, so a light path to his signature class. Move 8 + Canto.

Mastery detours: Death Blow [⛏ HARVEST — Brigand, Axe C boon] pushes Swift Strikes to ~+22 dmg — Axe is a boon for him, cheap harvest. Desperation [⛏ Cavalier, on path]. SKIP Darting Blow — Pegasus Knight is female-only.

5 skills: Lancefaire · Death Blow · Swift Strikes (combat art) · Hit +20 · Confidence (personal).

  • ✔ Swift Strikes + Lancefaire + Death Blow = one of the best player-phase tools in the game.
  • ✔ Move 8 + Canto; Lance/Riding boons mean near-zero friction to reach.
  • ✘ No flight; Confidence drops if he takes chip damage (avoid hazard tiles).
  • ✘ Ground-bound on flier-gated Maddening maps.

Build 2 — Wyvern Lord (飞龙领主)

Path: Noble → Fighter → Brigand → Wyvern Rider → Wyvern Lord. Cert: Axe A (boon) + Flying A (neutral — tutor). Trades Lancefaire for Move 8 flight, better growths, and Defiant Crit.

Mastery detours: Death Blow [⛏ HARVEST — Brigand, Axe C boon] already on the path — free. Pair Alert Stance+ with his Confidence for a premier dodge-tank.

5 skills: Axefaire · Death Blow · Alert Stance+ · Defiant Crit · Confidence (personal).

  • ✔ Move 8 + flight + Canto; Confidence (Hit/Avo +15) + Alert Stance+ = elite dodge-flier.
  • ✔ Best raw min-max ceiling per note.com.
  • ✘ Tutoring Flying A from neutral; loses Swift Strikes/Lancefaire.
  • ✘ Dodge reliance is risky on bad RNG turns.
