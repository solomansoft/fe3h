Byleth / 贝雷特 · 贝雷丝 — Build Menu V2

Player avatar / protagonist. Deploys every map; Crest of Flames sustain. Stats, growths, and proficiencies are identical for M and F — the ONLY mechanical difference is class access:
♀ Female Byleth → Pegasus/Falcon Knight, Gremory, Dark Flier, Valkyrie. ♂ Male Byleth → Grappler line, Hero, War Master, Dark Mage line. Both can be Wyvern Lord & Enlightened One (not gender-locked).
Proficiencies: Sword (boon), Brawl (boon), Authority (boon); budding Faith. Lance/Axe/Bow/Reason/Riding/Flying/HeavyArmor neutral — no banes.
Community trap — DO NOT use Mortal Savant: Sword A + Reason B+ is reachable, but the cache rates it poorly ("resource investment rarely justifies returns"). It is intentionally not used on any route below.
This playthrough uses 4 DISTINCT good builds — one per route (gender set by the route's romance/team). Each harvests Byleth's boon/neutral masteries.
Sources: GameRant / Game8 / GameFAQs / fe3h.com / Reddit (EN), GameWith / pegasusknight.com (JP), GamerSky (CN).

Build 1 — Enlightened One (神祖) AM · Normal · ♀

Path: auto-granted at end of Ch.10 — just equip it. Uses Sword + Gauntlets + Black/White magic; Swordfaire (+5), Terrain Res, all-round stat boosts; mastery Sacred Power (adjacent allies +3 dmg / −3 dmg taken). Cert: zero certification.

Mastery detours (boon/neutral-harvest): Death Blow [⛏ Brigand, Axe C — neutral] player-phase damage; Fiendish Blow [⛏ Mage, Reason C — neutral] only if you lean the magic side of the kit. Both off-boon-but-cheap; optional on Normal (R12: lean kit). His Sword/Brawl boons make the class's own faires free value.

5 skills: Swordfaire · Sacred Power · Hit+20 · White Magic Avo+20 · Death Blow.

  • ✔ Zero investment — combat-ready the moment it unlocks; maximizes the Sword of the Creator (Ruptured Heaven ranged art).
  • ✔ Sacred Power + Professor's Guidance buff adjacent allies — perfect for a low-stakes Normal run where Byleth babysits growing units.
  • ✔ Sword + Brawl boons + budding Faith all feed the hybrid sword/support kit.
  • ✘ Foot (5 move) — fine on Normal, would lag on Maddening.
  • ✘ Lower raw ceiling than the flier/War Master builds (intentional: Normal doesn't need it).

♀ on AM for the canonical Byleth × Dimitri romance arc. Not Mortal Savant (community trap).

Build 2 — War Master (战神) CF · Hard · ♂

Path: Brawler → Grappler → War Master (Master Seal). Cert: Brawl A (boon — free) + Axe A (neutral). Male-only class — this is why CF runs ♂ Byleth.

Mastery detours (boon/neutral-harvest): Death Blow [⛏ Brigand, Axe C — neutral] +6 Str on initiate, stacks with the class. Quick Riposte [⛏ War Master] its own mastery (enemy-phase follow-ups). Brawl boon means his Fistfaire/Crit chassis is cheap.

5 skills: Axefaire · Fistfaire · Crit+20 · Death Blow · Quick Riposte.

  • ✔ Top raw damage — Axefaire + Fistfaire + Crit+20 ride his Brawl boon; "arguably no better class" for pure damage on Byleth.
  • ✔ Half the path (Brawl) is a boon, so the only grind is Axe to A.
  • ✔ Hard difficulty rewards a fuller offensive kit (R12) without needing the full-Maddening flier optimization.
  • ✘ Foot (6 move) — more exposed than the wyvern/falcon shells.
  • ✘ Locks Byleth ♂ (no female-only classes), which is intentional for CF's romance/team.

♂ on CF (Byleth × Edelgard); War Master is male-locked so the gender choice and the build reinforce each other.

Build 3 — Falcon Knight (圣天马骑士) SS · Maddening · ♀ female-only

Path: Pegasus Knight (♀) → Falcon Knight. Cert: Sword C (boon) / Lance A (neutral) / Flying B+ (neutral). Female-only line — requires ♀ Byleth.

Mastery detours (boon/neutral-harvest): Darting Blow [⛏ Pegasus Knight, Lance C + Flying D — FEMALE-ONLY] sits directly on this path — free harvest (and only ♀ Byleth can ever get it). Death Blow [⛏ Brigand, Axe C — neutral] optional player-phase boost.

5 skills: Lancefaire · Darting Blow · Death Blow · Hit+20 · Alert Stance+.

  • ✔ 8 move + flight + Canto + Avo+10 + Defiant Avo — a mobile dodge-tank for Maddening SS maps.
  • ✔ Harvests the female-exclusive Darting Blow for guaranteed doubles, on her Sword/Lance path.
  • ✔ Forced ♀ by the Byleth × Seteth romance on SS — and that ♀ requirement unlocks exactly the Pegasus/Falcon line we want anyway.
  • ✘ Slightly less raw bulk than Wyvern Lord (more Spd/Res, less Str/Def).
  • ✘ Lance + Flying are neutral, so it's a real (if moderate) tutoring investment.

♀ on SS for Byleth × Seteth; the female lock is what makes Falcon Knight available, so the romance and the build are the same decision.

Build 4 — Wyvern Lord (飞龙领主) VW · Maddening · ♀

Path: Myrmidon → Brigand → Wyvern Rider → Wyvern Lord (not gender-locked). Cert: Lance C / Axe A (neutral) / Flying A (neutral). Heaviest tutoring of the four (Byleth has no native Flying), but the strongest chassis.

Mastery detours (boon/neutral-harvest): Death Blow [⛏ Brigand, Axe C — neutral] sits on the path — free harvest, cornerstone +6 Str. Hit+20 [⛏ Archer, Bow C — neutral] accuracy insurance for the axe build.

5 skills: Axefaire · Death Blow · Hit+20 · Defiant Crit · Alert Stance+.

  • ✔ The community-consensus strongest endgame chassis — 8 move + flight + Canto + Avo+10 + +4 Str/+4 Spd/+3 Def.
  • ✔ Byleth's Str/Spd growths + Crest of Flames sustain keep a fearless KO flier alive — "no real reason to use other classes over Wyvern Lord for Byleth."
  • ✔ Axe path harvests Death Blow for free.
  • ✘ Heaviest tutoring (Axe A + Flying A from neutral) — worth it on full Maddening.
  • ✘ Drops the Sword-relic synergy in favor of axe Might.

♀ on VW (Byleth × Claude); Wyvern Lord is not gender-locked, so gender is set purely by the romance while the build stays optimal.
