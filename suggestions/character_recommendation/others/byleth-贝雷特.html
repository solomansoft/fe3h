Byleth 贝雷特 / 贝雷丝 — Build Recommendation

Player avatar / protagonist. Stats, growths, Crest (Crest of Flames), proficiencies (Sword, Brawl, Authority; budding Faith) are IDENTICAL for M and F. The ONLY mechanical difference is class access: Female Byleth can become Pegasus Knight / Falcon Knight / Gremory / Dark Flier / Valkyrie; Male Byleth can become Grappler line / Dark Mage line / Hero / War Master. Both can become Wyvern Lord (not gender-locked).

Grounded in: others/protagonist/byleth-贝雷特.md, classes/ (Wyvern Lord, Falcon Knight, War Master, Enlightened One context, Gremory, Mortal Savant).

Class-Change Path (stack-ranked, to Master tier)

#1 — Wyvern Lord (飞龙领主) — BEST overall, BOTH genders

  • Why: The community-consensus strongest endgame chassis. 8 Mov + Flying, Canto, Avo +10, +4 Str/+4 Spd/+3 Def bonuses. Byleth's strong Str (45%) and Spd (45%) growths feed it perfectly, and Crest of Flames sustain keeps a flier alive. "There's no real reason to use other classes over Wyvern Lord for Byleth." [GameRant EN], [Reddit], [pegasusknight.com JP cache].
  • Path: Myrmidon/Soldier → (Brigand for Axe + Death Blow) → Wyvern Rider → Wyvern Lord.
  • ⚠️ Tutoring: Requires Lance C / Axe A / Flying A. Byleth has no Flying proficiency natively (Flying is Jeritza's talent, not Byleth's), so this needs heavy Axe + Flying tutoring/training. Worth it.

#2 — Enlightened One (神祖) — Byleth's unique class, low-effort default, BOTH genders

  • Why: Auto-unlocked at end of Ch.10; needs zero certification investment. Uses Sword + Gauntlets + Black/White magic, Swordfaire (+5), Terrain Resistance, all-round stat boosts. Mastery Sacred Power buffs adjacent allies (+3 dmg / −3 dmg taken). "Best choice for our main character as a sword user." [Game8 EN]. Foot class (5 Mov) is its only real downside vs Wyvern Lord.
  • Path: It is granted automatically; just equip it. Great as the always-available fallback or for a sword-and-support hybrid.

#3 — War Master (战神) — MALE Byleth only, raw damage

  • Why: Top Str growth bonus, Axefaire + Fistfaire + Crit +20, mastery Quick Riposte. With Byleth's Brawl strength this is "arguably no better class" for pure damage. Foot (6 Mov), more exposed than Wyvern Lord. [GameRant EN], [GameWith JP cache].
  • ⚠️ Tutoring: Axe A + Brawl A. Byleth has Brawl strength so half is free; Axe needs work.

#4 — Falcon Knight (圣天马骑士) — FEMALE Byleth only, mobile/avoid

  • Why: Female-exclusive; 8 Mov + Flying, +5 Spd, Avo +10, Canto, Defiant Avo mastery. "Caters to a lot of her natural abilities." Slightly worse raw bulk than Wyvern Lord but higher Spd/Res. [GameRant EN], [pegasusknight.com JP cache].
  • ⚠️ Tutoring: Sword C / Lance A / Flying B+. Sword is free; Lance + Flying need training.

Alternatives / niche

  • Mortal Savant (魔剑士): possible (Sword A + Reason B+) since Byleth can use magic, but community + cache rate it poorly ("resource investment rarely justifies returns" [pegasusknight.com]). Avoid.
  • Gremory (格莫瑞, F only): only if going full magic — wastes Byleth's physical growths. Not recommended.
  • GamerSky CN: "generally drop magic, focus physical output" for Byleth. Aligns with Wyvern Lord/War Master picks. [GamerSky CN cache].

Mastery detours / skills to grab

Classes to certify into ONLY to fill the class-mastery bar and learn the permanent mastery ability, then leave — these are NOT part of the final class path above. - Brigand → Death Blow (+6 Str when this unit initiates combat) — top priority; the cornerstone offensive skill for every physical build (Wyvern Lord / War Master). - Pegasus Knight (F) / Wyvern Rider → Darting Blow (+5 Spd when initiating) — push her/his Spd lead for guaranteed doubles. (Darting Blow is the Pegasus Knight mastery; F!Byleth can grab it directly, M!Byleth substitutes other Spd skills.) - Archer → Hit +20 (+20 Hit) — accuracy insurance for the Wyvern/War Master damage builds. - Mage → Fiendish Blow (+6 Mag when initiating) — only if you run an Enlightened One / magic-hybrid build; skip for pure physical.

Training Focus

  • Prioritize Axe + Flying if aiming Wyvern Lord (the only major detour, since Byleth lacks Flying natively). Tutor in the seminar/instruction phases.
  • Keep Sword maxed (free strength) and Brawl up (free strength) for combat-art flexibility and War Master (M).
  • Authority strength → push to A/A+ for the best battalions and gambits; Byleth deploys every map, so Authority XP accrues naturally.
  • Budding Faith → unlocks White Magic Avoid +20 (great dodge skill); a few Faith levels also give Heal/Nosferatu utility for Enlightened One.
  • Get Death Blow (Brigand mastery, +6 Str on initiate) and Darting Blow (Pegasus/Brigand area) early — cornerstone offensive skills.

Supports / Pairings (Byleth can S-support a huge cast)

  • Gameplay synergy: Pair with a healer/buffer adjacent so Crest of Flames + the partner stay topped up; Byleth's own Professor's Guidance(+) feeds adjacent allies +1.2x EXP and +2 dmg — keep growing units beside Byleth.
  • Romance (story-favorite picks, per Reddit):
  • F!Byleth → Dimitri (most cited), Claude, Dorothea (f/f favorite), Ferdinand.
  • M!Byleth → Bernadetta (most cited), Edelgard, Dorothea.

Weapon / Item / Battalion / Skill Loadout

  • Wyvern Lord build (best):
  • Weapons: a forged Brave Sword or Killer Sword + a good Axe (Brave Axe / Sword of Begalta if available); Sword of the Creator (Byleth's relic) for ranged whip attacks.
  • Combat Arts: Sword of the Creator's ranged art (Ruptured Heaven), plus Axe arts (Smash) when in Wyvern.
  • 5 equipped skills: Swordfaire, Death Blow (or Darting Blow), Hit +20, Alert Stance+ (Avo tank), Death Blow/Defiant Crit (Wyvern mastery).
  • Battalion: a high-Phys-Atk / high-Authority battalion; Gambit: Onslaught or a Retribution-type for enemy-phase. [Reddit cache rates Retribution top gambit.]
  • Enlightened One build: Sword of the Creator + a Reason/Faith tome; skills Swordfaire, Sacred Power (mastery), Hit +20, White Magic Avoid +20, Death Blow.
  • Key items: Sword of the Creator (auto), Goddess Ring / stat boosters on Byleth since always deployed.
